!!! THIS FILE IS FOR USERS WHO WANT TO BUILD BINARIES FROM THE SOURCE CODE FOR THEIR OWN PLATFORM. WE RECOMMEND THAT IF A PREBUILT BINARY EXISTS FOR YOUR PLATFORM, USE IT. !!!

If you are on a platform that we do not make prebuilt binaries for, you will have to build Cobalt yourself to use it.

You will need to install Rust, which includes Cargo. You can look up how to do this if you aren’t sure.

After, change into this directory and run cargo build –release

This should automatically build an optimized binary executable for your system. The produced binary should be placed in the path “target/release” relative to the root of the project. After that, you can enjoy using Cobalt!
