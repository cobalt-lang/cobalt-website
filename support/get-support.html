Support

Experiencing bugs or security issues with the website, or with the language?

You may contact us at our email.

Expect response times to be within 24 hours. If it is longer, please do not send another ticket, simply wait for a response.

You may be asked questions regarding your issue. Please answer them to help us slove this problem for you.

If you know what seems to be causing the issue and want to provide a fix, you are welcome to.


Please do not email for unrelated issues or issues regarding your code. If you don't know how to do something in Cobalt, check the docs.

You may email us regarding:

  • Bugs with the website
  • Bugs with the programming language
  • Security issues with the website
  • Security issues with the programming language

You may not email us regarding:

  • Help with your code
  • Unrelated topics

An email should be formatted something like this:


Subject: Bug Report with Website

Body:

Hello. The website is having bugs on my platform. I tried to access a certain page which loaded incorrectly.
The page seemed to be missing a stylesheet.

Link: https://cobalt-lang.com/badpage.html
Platform: Windows 10 64-bit
Browser: Chrome
